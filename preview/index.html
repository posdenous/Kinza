Kinza Berlin - Application Previews

This page provides links to live HTML previews for various user flows and administrative tools. These are used for rapid prototyping, UI testing, and stakeholder reviews.

User-Facing Previews (Parent/Guardian)

  • Parent Onboarding Flow

    Simulates the complete first-time user experience, including language selection, privacy consent, and child profile creation.

  • Search & Discovery

    Prototypes the core event discovery experience, showing how users can search for and find activities.

  • Map View

    Visualizes how events are displayed on a map interface for geographical discovery.

  • Saved Events

    Shows the screen where users can view events they have saved for later.

  • Privacy Settings

    Focuses on the user-facing privacy controls, including consent management and data deletion.

Admin & Moderator Previews

  • Content Moderation Dashboard

    An internal tool for administrators to review, approve, or reject user-generated content that has been flagged.

  • Admin Dashboard

    Administrative interface for managing content, users, and settings.

  • Organiser Dashboard

    Dashboard for event organizers to manage their events and track attendance.
